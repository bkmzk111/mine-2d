### Installation
Currently the project can be only compiled from source using `cmake`. Was only tested on:
- Clang 21.1.6 x86_64-pc-linux-gnu `cmake` toolchain
- Arch Linux
- KDE Plasma environment
#### Supposingly works on any Linux distribution and might compile on Windows
